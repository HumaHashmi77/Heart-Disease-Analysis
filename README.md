# Heart-Disease-Analysis
Data analysis and predictive modeling of heart disease using Python to identify risk factors and support early diagnosis.

# ❤️ Heart Disease Analysis & Prediction using KNN

## 📌 Overview
Heart disease is one of the leading causes of death worldwide. Early detection and understanding of risk factors can significantly improve patient outcomes.
This project focuses on analyzing a heart disease dataset to identify important health indicators and build a machine learning model to predict the likelihood of heart disease.

## 🎯 Objectives
- Perform data cleaning and preprocessing
- Explore patterns and relationships in the dataset
- Identify key risk factors for heart disease
- Build a machine learning model (KNN) for prediction
- Evaluate model performance

## 📊 Dataset
The dataset contains patient medical information such as:
- Age
- Sex
- Chest pain type
- Blood pressure
- Cholesterol levels
- Maximum heart rate
- Other clinical features

Target variable:
- `target` → Indicates presence (1) or absence (0) of heart disease

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## 🔧 Data Preprocessing
- Renamed columns for clarity
- Checked and handled missing values
- Converted data types where necessary
- Cleaned dataset for analysis

## 📈 Exploratory Data Analysis (EDA)
- Visualized feature distributions using histograms
- Used heatmaps to analyze correlations
- Explored relationships between features and target variable

### 🔍 Key Insights:
- Older individuals are more likely to have heart disease
- Certain chest pain types show strong correlation with heart disease
- Cholesterol and heart rate patterns provide useful indicators
- Some features have stronger predictive power than others

## 🤖 Machine Learning Model: K-Nearest Neighbors (KNN)

### 📌 Why KNN?
KNN is a simple and effective classification algorithm that predicts the class of a data point based on its nearest neighbors.

## 📌 Conclusion
- Data analysis revealed important health factors linked to heart disease
- The KNN model successfully predicts the likelihood of heart disease
- This approach can support early diagnosis and decision-making in healthcare
